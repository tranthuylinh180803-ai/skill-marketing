# Shared CTAs Library

Thư viện CTA (Call to Action) theo mục tiêu — copy và tùy biến.

---

## CTA theo mục tiêu

### Engagement (Tương tác)
```
Comment:
- "Comment '[từ khóa]' để nhận [guide/template/checklist]"
- "Bạn đồng ý không? Comment bên dưới"
- "Tag người bạn nghĩ cần đọc cái này"
- "Comment 'YES' nếu bạn đang gặp vấn đề này"

Save:
- "Save bài này — bạn sẽ cần nó sau"
- "Lưu lại để xem khi cần"

Share:
- "Share nếu bài này hữu ích với bạn"
- "Gửi cho ai đang cần đọc điều này"
```

### Growth (Tăng follower)
```
- "Follow để không bỏ lỡ phần tiếp theo"
- "Follow nếu bạn muốn học thêm về [chủ đề]"
- "Theo dõi để nhận thêm tips về [chủ đề] mỗi ngày"
```

### Lead generation
```
- "DM tôi '[từ khóa]' để biết thêm chi tiết"
- "Link ở bio — nhấn để nhận [offer miễn phí]"
- "Đăng ký miễn phí tại link trong bio"
- "Nhắn tin [từ khóa] để nhận tư vấn miễn phí"
```

### Direct conversion
```
- "Mua ngay — chỉ còn [số] suất / [thời gian]"
- "Dùng code [CODE] để được giảm [X]%"
- "Đặt hàng tại link bio — ship trong 24h"
- "Nhắn tin 'ORDER' để đặt hàng ngay"
```

## CTA theo nền tảng

### TikTok
```
- "Follow để xem phần 2" (series)
- "Stitch video này nếu bạn đồng ý"
- "Link ở bio nha" (không thể link trực tiếp)
```

### Instagram
```
Feed: "Lưu bài này 📌"
Story: [Swipe up / Link sticker]
Reels: "Follow để không miss phần tiếp"
```

### Facebook
```
- "Comment bên dưới nhé!"
- "Share cho bạn bè cùng xem"
- "Nhắn tin page để được hỗ trợ"
```

### LinkedIn
```
- "Repost nếu bạn thấy hữu ích" (không nói "like")
- "Comment quan điểm của bạn bên dưới"
- "Connect nếu bạn làm trong [ngành]"
```

### Email
```
Subject line CTA: "Claim your spot →", "Open before [ngày]"
Body CTA: Single button — "Get started free", "Book a call", "Download now"
P.S. CTA: Backup CTA nhẹ hơn ở cuối email
```

## CTA theo funnel stage

### TOFU (Awareness)
```
Low commitment: "Đọc thêm", "Xem thêm", "Follow"
```

### MOFU (Consideration)
```
Medium commitment: "Download free guide", "Đăng ký webinar miễn phí", "Xem demo"
```

### BOFU (Conversion)
```
High commitment: "Mua ngay", "Đăng ký dùng thử", "Đặt lịch tư vấn"
```

---

## Quy tắc viết CTA hiệu quả
1. **1 CTA duy nhất** — nhiều CTA = không CTA nào được thực hiện
2. **Verb mạnh** ở đầu: "Get", "Start", "Download", "Nhận", "Đăng ký"
3. **Benefit rõ ràng**: "Download free guide" tốt hơn "Click here"
4. **Urgency** khi phù hợp: "Chỉ còn 24h", "Giới hạn X suất"
5. **Friction thấp**: CTA TOFU phải dễ hơn CTA BOFU
