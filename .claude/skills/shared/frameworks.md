# Shared Frameworks — Marketing

Thư viện framework dùng chung cho tất cả skills.

---

## Copywriting Frameworks

### AIDA
```
Attention → Interest → Desire → Action
Gây chú ý → Tạo hứng thú → Khơi gợi mong muốn → Kêu gọi hành động
```

### PAS
```
Problem → Agitate → Solution
Nêu vấn đề → Khuếch đại nỗi đau → Đưa ra giải pháp
```

### PASTOR
```
Problem → Amplify → Story → Transformation → Offer → Response
```

### FAB
```
Feature → Advantage → Benefit
"Sản phẩm có [Feature], điều này giúp [Advantage], vì vậy bạn nhận được [Benefit]"
```

## Persuasion Frameworks

### Feel / Felt / Found (Objection handling)
```
"Tôi hiểu bạn CẢM THẤY như vậy.
Nhiều khách hàng của chúng tôi cũng ĐÃ CẢM THẤY thế.
Nhưng sau khi dùng, họ NHẬN RA rằng..."
```

### Pain / Solution / Outcome
```
[Pain]: Khách đang gặp vấn đề gì?
[Solution]: Sản phẩm giải quyết ra sao?
[Outcome]: Kết quả sau khi dùng?
```

### Before / After / Bridge
```
Before: Cuộc sống trước khi có sản phẩm
After: Cuộc sống sau khi có sản phẩm
Bridge: Sản phẩm là cầu nối
```

## Content Strategy Frameworks

### 80/20 Content Mix
```
80% nội dung giá trị:
- 40% Education (dạy điều gì đó)
- 25% Entertainment (giải trí, relatable)
- 15% Inspiration (truyền cảm hứng)

20% nội dung bán hàng:
- Promotion, offer, CTA trực tiếp
```

### Content Pillars (4 trụ cột)
```
Pillar 1: Authority (chứng tỏ chuyên môn)
Pillar 2: Community (kết nối với khán giả)
Pillar 3: Product (giới thiệu sản phẩm)
Pillar 4: Culture (văn hóa thương hiệu)
```

### TOFU / MOFU / BOFU
```
Top of Funnel (Awareness): Education, entertainment, broad content
Middle of Funnel (Consideration): Case study, comparison, demo
Bottom of Funnel (Conversion): Offer, testimonial, urgency
```

## Video Frameworks

### Short-form retention loop
```
Hook (0-3s) → Curiosity gap → Fast value → Pattern interrupt → CTA
```

### Story Arc (3 Act)
```
Act 1 (20%): Setup — nhân vật, bối cảnh, trạng thái "trước"
Act 2 (60%): Conflict — vấn đề, hành trình tìm giải pháp
Act 3 (20%): Resolution — kết quả, transformation, CTA
```

## Sales Frameworks

### SPIN Selling
```
Situation: "Hiện tại bạn đang xử lý X như thế nào?"
Problem: "Điều đó có gây ra Y không?"
Implication: "Nếu không giải quyết, điều đó ảnh hưởng thế nào đến Z?"
Need-payoff: "Nếu có giải pháp giúp kết quả, nó có giá trị không?"
```

### Challenger Sale
```
Teach (dạy điều họ chưa biết) →
Tailor (cá nhân hóa cho tình huống của họ) →
Take control (dẫn dắt cuộc trò chuyện)
```
